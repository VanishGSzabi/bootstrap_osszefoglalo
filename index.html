Rövid bemutató és csatolás

Mi az a Bootstrap?

A Bootstrap egy nyílt forráskódú keretrendszer weboldalakhoz, mely CSS és JavaScript bővítéseket tartalmaz. Ezekhez tartoznak az alapvető szépítések (kép formázása), előre elkészített elemek (képcsúszka) vagy egy keret struktúra a weboldal reszponzivitására.

Mire jó?

Először is, könnyen és egyszerűen lehet vele a weboldalunkat formálni. Több esetben egyszerűen csak egy class="xyz" fog kelleni. Ráadásul ingyenes, és tágas a dokumentációja is, mindenre van leírás.

Bootstrap beillesztése

  1. Lokálisan alkalmazás
    1. Töltsük le innen az állományt
    2. Csomagoljuk ki a weboldalunk mappájába
    3. Csatoljuk a HTML fejlécében:
      • link rel="stylesheet" href="bootstrap.css"
      • script src="bootstrap.js"
  2. Online repo-ról alkalmazás
    • Csatoljuk a HTML fejlécében:
      • link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" crossorigin="anonymous"
      • script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"

Néhány Bootstrap elemhez a jQuery JavaScript állomány is szükséges! Ez itt elérhető.

Elemek szépítése

Táblázat

A HTML táblázatokban a table - mint egész táblázat - és tr - mint a táblázat sorait - lehet "díszíteni" a Bootstrap által definiált osztályokkal.

Table formázása:
  • class="table" -> Bootstrap alap stílusa
  • class="table-striped" -> minden páratlan sor szürke háttérszínű
  • class="table-bordered" -> hozzáad egy vékony szegélyt a táblázathoz
  • class="table-hover" -> kurzorral rámenve a háttérszín szürke lesz
  • class="table-sm" -> nincsen cellamargó
Demonstráció + tr / table hátterek

Képek

A képek elhelyezésében, formázásában és méretezésében is segít a Bootstrap. Ismét, csak osztályokat kell hozzáadni egy meglévő képünkhöz és ennyi.

Például:
  • mx-auto d-block -> a kép középre lesz helyezve
  • rounded -> 6* px-el kerekítve lesz a kép sarka
  • rounded-circle -> teljes kerekítés
  • img-thumbnail -> vékony szegély a kép körül
  • img-fluid -> a kép mérete igazodik a körülvévő kerethez
Demonstráció

Gombok

A Bootstrap által szép és modern megjelenésű gombokat tudunk alkalmazni különböző színben, méretben és még mi más. A btn osztállyal lehet Bootstrap-es formázást alkalmazni.

Méretek:
  • btn-lg -> nagy méretű gomb
  • btn-sm -> kis méretű gomb
