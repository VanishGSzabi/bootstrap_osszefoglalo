Mi az a Bootstrap?

A Bootstrap egy nyílt forráskódú keretrendszer weboldalakhoz, mely CSS és JavaScript bővítéseket tartalmaz. Ezekhez tartoznak az alapvető szépítések (kép formázása), előre elkészített elemek (képcsúszka) vagy egy keret struktúra a weboldal reszponzivitására.

Mire jó?

Először is, könnyen és egyszerűen lehet vele a weboldalunkat formálni. Több esetben egyszerűen csak egy class="xyz" fog kelleni. Ráadásul ingyenes, és tágas a dokumentációja is, mindenre van leírás.

Bootstrap beillesztése

  1. Lokálisan alkalmazás
    1. Töltsük le innen az állományt
    2. Csomagoljuk ki a weboldalunk mappájába
    3. Csatoljuk a HTML fejlécében:
      • link rel="stylesheet" href="bootstrap.css"
      • script src="bootstrap.js"
  2. Online repo-ról alkalmazás
    • Csatoljuk a HTML fejlécében:
      • link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" crossorigin="anonymous"
      • script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
